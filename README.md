# Banking Transaction Management System

Full Stack Java backend application built using Spring Boot and REST APIs.

## Features
- Account management
- Fund transfer functionality
- Java backend using OOP principles
- RESTful API design
- Basic transaction validation

## Technologies Used
- Java
- Spring Boot
- REST APIs
- JDBC (conceptual)
- MySQL (conceptual)
- Postman

## API Endpoints
- GET /api/accounts
- POST /api/accounts/transfer
